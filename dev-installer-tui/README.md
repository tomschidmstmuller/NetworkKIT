# Dev Installer TUI

A terminal user interface for installing development tools on Linux with support for multiple package managers.

## Features

- **Multi-package manager support**: nala, apt, pacman, yay (AUR)
- **TUI interface** with categories and package selection
- **Real-time installation progress**
- **System update functionality**
- **Auto-detection** of Linux distribution
- **Checkbox selection** for multiple packages

## Supported Distributions

- Ubuntu/Debian (apt/nala)
- Arch Linux (pacman/yay/AUR)

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run:
```bash
python main.py
```
